
# Node Cowsay
Node Cowsay example

## Installation

- Install GitHub CLI, gh is available via [Homebrew](https://brew.sh/index_es)
```bash
brew install gh
```

-

## Usage

```python
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```
